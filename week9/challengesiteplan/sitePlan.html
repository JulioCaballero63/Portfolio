Challenge Site Plan

Site Purpose

To build a web application to explore and apply the things you have learned thus far.

Target Audience

Users who want to find out the current weather and a seven day forecast for cities in USA. The site will also show users coming events and other useful information about the selected city.

Source

The data will be retrieved from openweathermap.org/api

Color Scheme

Typography

The font family for h1, h2, h3 will be "Roboto Slab" or serif if font not supported.

The fort family for paragraphs will be "Open Sans" or san-serif if font not supported.
